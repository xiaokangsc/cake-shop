🎯 第三阶段 Day 46-90 · 10大办公任务 × 全平台实测

这个阶段是全网独有的差异化内容：同一个任务，让所有AI平台各做一遍，横向对比谁最强

📊 测试方法论

每个任务统一标准：
① 同一段提示词（中文）→ 发给所有平台
② 记录：生成速度 · 质量 · 是否需要修改 · 修改几次 · 总耗时
③ 最后输出：「这个任务，第1名是XXX，原因是XXX」
④ 每完成一个任务出一期对比视频（爆款潜质）

10大办公任务（从简单到复杂）

📝 任务一：写一条朋友圈推广文案   ⭐ 入门
提示词：「我开了一家蛋糕店，帮我写一段发朋友圈的文案，推广巴斯克芝士蛋糕，¥59一个，要求轻松活泼，100字以内」
测试平台：DeepSeek · Kimi · 豆包 · 通义 · 文心 · 讯飞 · ChatGPT · Claude
视频时长：60-90秒（快速对比8个结果并排展示）
📊 任务二：用Excel公式处理销售数据   ⭐⭐ 简单
提示词：「下面是蛋糕店一周销售数据（粘贴数据），帮我用公式算出：每天总销售额、最畅销产品、平均客单价、环比增长率」
测试平台：DeepSeek · Kimi · 豆包 · ChatGPT · Claude · 通义
视频亮点：哪个AI公式直接能用？哪个算错了？
📑 任务三：生成一份10页的PPT大纲+内容   ⭐⭐ 简单
提示词：「我要做一个'2026年蛋糕店营销计划'的PPT，10页，包括：市场分析、产品策略、定价方案、推广渠道、预算分配。请给出每页标题和要点」
测试平台：全部聊天AI + Gamma
视频亮点：同需求→8个PPT大纲→哪个最专业？
🖼 任务四：生成一张促销海报   ⭐⭐⭐ 中等
提示词（中文）：「设计一张蛋糕店520促销海报，粉色系，主标题'520 甜蜜告白'，副标题'全场蛋糕8折'，要有蛋糕元素和浪漫氛围」
测试平台：即梦 · 可灵 · Midjourney · DALL-E · LiblibAI
视频亮点：5张海报并排对比，粉丝投票选最美
🎬 任务五：生成一条15秒短视频   ⭐⭐⭐ 中等
提示词：「蛋糕店宣传视频，15秒，展示巴斯克蛋糕切开的流心画面，配上文字'一口沦陷'」
测试平台：可灵视频 · Runway · Pika · 剪映AI
视频亮点：4个AI生成的视频并排播放，太震撼了
📄 任务六：根据录音生成会议纪要   ⭐⭐⭐ 中等
任务：放一段模拟录音（你录一段蛋糕店开会讨论），让AI转文字+总结
测试平台：讯飞星火 · 剪映 · 豆包 · Kimi · 通义
视频亮点：语音转文字谁最准？总结谁最简洁？
💻 任务七：写一个完整的产品展示网页   ⭐⭐⭐⭐ 进阶
提示词：「帮我做一个蛋糕店的产品展示页面，包含：顶部Banner轮播、6个产品卡片（图片+名称+价格+购买按钮）、底部联系方式。纯HTML/CSS/JS，要好看」
测试平台：Claude Code · Codex CLI · Cursor · Windsurf · Cline · 小龙虾AI · MarsCode
视频亮点：7个AI各写出什么网页？并排对比效果炸裂
🤖 任务八：搭建一个AI客服机器人   ⭐⭐⭐⭐ 进阶
任务：同一个客户常见问题文档，导入后测试问答效果
测试平台：Coze · Dify · FastGPT · 文心智能体 · 小龙虾AI
视频亮点：哪个平台搭的AI客服最聪明？
🛒 任务九：做一个完整的电商小程序页面   ⭐⭐⭐⭐⭐ 复杂
提示词：「做一个蛋糕店微信小程序首页：轮播Banner+分类筛选+产品列表+底部导航栏。用uni-app框架，要能直接运行的代码」
测试平台：Claude Code · Codex CLI · Cursor · Windsurf · Cline
视频亮点：5个AI各写一个小程序页面——效果天差地别！分上下集
🚀 任务十：从零到部署上线一个完整网站   ⭐⭐⭐⭐⭐ 终极
