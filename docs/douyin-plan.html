阶段一 · Day 1-10 · 10大编程Agent安装实测

这10天每天装一个全网最火的AI编程工具，同一个难度递增的编码任务发给它，记录安装过程、生成质量、token消耗。10天后排出从夯到拉的完整排名。

Day 1｜难度 ⭐
Claude Code
任务：写一个自我介绍HTML页面（包含姓名/职业/技能/联系方式）
安装重点：①获取DeepSeek API Key（platform.deepseek.com注册）→ ②VS Code安装Claude Code插件 → ③settings.json配置ANTHROPIC_BASE_URL和AUTH_TOKEN → ④模型必须选deepseek-v4-pro[1m]
常见坑：Base URL填错端口/模型名不写[1m]后缀会报错/Auth Token忘记Bearer前缀
预计token：~5K | 预计耗时：安装30min + 任务测试15min
编程Agent免费API
Day 2｜难度 ⭐
Codex CLI
任务：写一个带加减乘除功能的计算器网页
安装重点：①安装Node.js 18+ → ②npm install -g @openai/codex → ③配置OPENAI_API_KEY环境变量 → ④终端运行codex命令
常见坑：国内需用API中转/Node版本过低/Git未安装
预计token：~8K | 预计耗时：安装20min + 测试15min
编程Agent需API付费
Day 3｜难度 ⭐
小龙虾AI
任务：创建一个自动问答Bot（输入问题→匹配预设答案→输出回复）
安装重点：①浏览器打开xiaolongxia.cn注册 → ②新建Agent→配置知识库→设置触发关键词 → ③测试问答效果
常见坑：知识库导入格式要求JSON/免费版有调用次数限制
预计token：免费额度 | 预计耗时：注册5min + 搭建20min
Agent平台国内免费
Day 4｜难度 ⭐⭐
WorkBuddy
任务：写一个带渐变背景+圆角卡片+价格标签的产品展示卡片
安装重点：①下载WorkBuddy客户端安装 → ②打开后自动识别Claude Code集成 → ③配置DeepSeek API（同Day1）→ ④使用Skills市场安装插件
常见坑：Skills市场插件下载失败需重试/首次启动配置时间较长
预计token：~5K | 预计耗时：安装25min + 测试15min
IDE集成免费
Day 5｜难度 ⭐⭐
Cursor
任务：写一个自适应手机+电脑的响应式导航栏（含下拉菜单）
安装重点：①官网下载Cursor安装 → ②GitHub账号登录 → ③打开Composer模式（Ctrl+I）→ ④输入任务需求
常见坑：免费版每月的2000次补全用完即停/Pro版$20月
预计token：免费额度内 | 预计耗时：安装10min + 测试20min
AI编辑器有限免费
Day 6｜难度 ⭐⭐
Windsurf
任务：写一个包含首页/关于/联系三个页面的简单网站框架
安装重点：①官网下载Windsurf → ②注册Codeium账号 → ③使用Cascade模式进行多文件编辑 → ④对比Cursor的Composer
常见坑：免费额度限制/多文件编辑时偶尔索引失败
预计token：免费额度内 | 预计耗时：安装10min + 测试25min
AI编辑器有限免费
Day 7｜难度 ⭐⭐⭐
Cline
任务：写一个带表单验证的用户注册页面（含邮箱/手机/密码校验）
安装重点：①VS Code扩展商店搜索Cline安装 → ②配置API Key和模型 → ③设置auto-approve权限范围 → ④开启MCP工具连接
常见坑：API消耗速度极快（一次复杂任务可能用掉$3-5）/需手动设置预算上限防止超额
预计token：~20K | 预计耗时：安装15min + 测试25min
编程AgentAPI消耗大
Day 8｜难度 ⭐⭐⭐
Aider
任务：在一个已有HTML项目中添加暗黑模式切换功能和localStorage记忆
安装重点：①安装Python3 → ②pip install aider-chat → ③配置API_KEY环境变量 → ④终端进入项目目录运行aider
常见坑：需在git仓库中运行/终端操作门槛较高/中文文件路径可能乱码
预计token：~15K | 预计耗时：安装15min + 测试20min
编程Agent需Python
Day 9｜难度 ⭐⭐
通义灵码
任务：写一个使用Chart.js的数据可视化柱状图和饼图组合页面
安装重点：①VS Code扩展商店搜索"通义灵码"安装 → ②阿里云账号登录 → ③Alt+\ 触发代码补全 → ④Ctrl+Shift+I 打开对话
常见坑：仅支持VS Code和JetBrains/偶尔智能补全延迟
预计token：免费 | 预计耗时：安装5min + 测试15min
编程插件完全免费
