🎯 第三阶段 Day 39-90 · 10大任务 × 全平台实测

全网独家内容：同一个任务发给所有AI平台，横向对比——不是比谁能做，而是谁做得好、做得快、做得对

每个任务统一标准：同一段中文提示词→发给所有平台→记录：速度/质量/需要改几次/最终能不能用 → 输出排名+优缺点分析

📝 任务一：写文案 ⭐ 入门
同一个需求：「帮我写一段推广文案，轻松活泼，100字左右」
测试平台：DeepSeek/Kimi/豆包/通义/文心/讯飞/ChatGPT/Claude
视频：8个结果并排展示，分析谁的语言最自然、谁最啰嗦、谁最AI味
📊 任务二：Excel数据处理 ⭐⭐ 简单
同一组数据发给所有AI：「帮我写公式算出总量、平均值、增长率、最高最低值」
测试平台：DeepSeek/Kimi/豆包/ChatGPT/Claude/通义
视频：哪个AI公式复制粘贴就能用？哪个算错了？哪个步骤太复杂？
📑 任务三：PPT大纲生成 ⭐⭐ 简单
同一个PPT需求发给8个AI：「10页PPT，包含背景、方案、执行、预算、总结」
测试平台：DeepSeek/Kimi/豆包/ChatGPT/通义/Gamma
视频：对比结构的逻辑性、内容的深度、是否可直接用
🖼 任务四：AI生图 ⭐⭐⭐ 中等
同一句中文提示词发给5个平台，对比出图效果
测试平台：即梦/可灵/Midjourney/DALL-E/LiblibAI
视频：5张图并排，分析「谁理解中文最好」「谁最逼真」「谁最有质感」
🎬 任务五：AI生成短视频 ⭐⭐⭐ 中等
同一段文字描述，让AI视频平台各生成一条视频
测试平台：可灵视频/Runway/Pika/剪映AI
视频：4个结果并排播放，分析谁最流畅、谁最自然、谁最假
📄 任务六：语音转文字+总结 ⭐⭐⭐ 中等
同一段录音发给5个AI：「转成文字+总结要点+提取待办事项」
测试平台：讯飞星火/剪映/豆包/Kimi/通义
视频：谁转得最准？谁总结得最好？谁漏了关键信息？
💻 任务七：写一个完整的网页 ⭐⭐⭐⭐ 进阶
同一个需求：「做一个信息展示网页，包含导航、Banner、内容卡片、底部。要好看能直接用」
测试平台：Claude Code/Codex CLI/Cursor/Windsurf/Cline/小龙虾/MarsCode
视频：7个AI写的网页并排展示——核心不是能不能写，而是谁写的更专业、代码更整洁、更少bug
🤖 任务八：搭建自动化工作流 ⭐⭐⭐⭐ 进阶
同一个需求：「搭建一个自动回复系统：用户输入问题→匹配知识库→输出答案」
测试平台：Coze/Dify/FastGPT/文心智能体/小龙虾
视频：分析搭建难度、调试体验、最终效果、可扩展性
🏗 任务九：做一个完整应用 ⭐⭐⭐⭐⭐ 复杂
同一个复杂需求：「做一个多页面的Web应用，包含列表页、详情页、表单页，要完整可运行」
测试平台：Claude Code/Codex CLI/Cursor/Windsurf/Cline
视频：5个平台各做一个应用——核心评测各平台处理复杂项目的能力：上下文管理、多文件协同、报错修复能力。分上中下三集。
🚀 任务十：从零到部署全流程 ⭐⭐⭐⭐⭐ 终极
终极测试：给同一个需求，让各平台从写代码→调试→部署→上线完整跑一遍
测试平台：Claude Code vs Codex CLI vs Cursor
视频：90天收官——全面复盘各平台优劣势：哪个适合新手、哪个适合专业开发、哪个性价比最高。公布完整对比排名。
