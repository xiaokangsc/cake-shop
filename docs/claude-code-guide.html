🤖Claude Code 完全使用教程

从零到精通 · 许仁康整理 · 2026年6月

一、什么是 Claude Code

Claude Code 是一个AI 编程助手，你对他说话，他帮你：

能做什么举例
💻 写网站/小程序「帮我做一个蛋糕店在线商城」
📝 改代码「把这个按钮颜色改成红色」
🔧 部署上线「帮我部署到 GitHub Pages」
🖼 处理图片「把这些产品图裁剪一下去掉黑边」
📊 数据处理「帮我录入这7款新产品」
📄 生成文档「帮我做一份 PDF 产品目录」

二、你的 Claude Code 是怎么搭的

1安装 WorkBuddy（国内版 Claude Code IDE）
2配置 DeepSeek API：在 .claude/settings.json 里填了 DeepSeek 的 API 地址和密钥
3模型选择：主模型 deepseek-v4-pro[1m]（1M 上下文窗口），子代理用 deepseek-v4-flash
4Skills 市场：WorkBuddy 自带 200+ 技能，包括腾讯文档、企业微信、浏览器控制等

三、怎么在手机上使用

📱 方案一：Claude 官方 App（推荐）

1iPhone 切到美区 App Store / Android 下载 Claude App
2登录你的 Anthropic 账号（需要外区手机号或 Google 登录）
3直接在聊天框里对话，和电脑端 Claude Code 功能一样
需要外区 Apple ID 或 Google Play 账号才能下载

📱 方案二：手机浏览器

1手机打开 https://claude.ai
2登录后在网页版直接使用
网页版功能完整，不用下载 App，最方便

📱 方案三：远程控制电脑

1电脑和手机都安装 ToDesk 或 向日葵
2手机远程连到电脑桌面
3在手机上操作电脑里的 Claude Code，随时随地写代码

四、Claude Code 常用命令

在 VS Code 或终端里使用：

# 初始化项目
/init

# 代码审查
/code-review

# 安全审查
/security-review

# 简化代码
/simplify

# 深度研究
/deep-research 主题描述

# 配置更新
/update-config

五、Skills（技能）怎么用

你的 WorkBuddy 自带这些技能，直接对话就能触发：

Skill用途怎么触发
deep-research深度研究，联网搜索「帮我研究一下抖音蛋糕店运营方法」
code-review代码审查「帮我检查刚才的代码有什么问题」
claude-apiAPI 参考「Claude API 最新价格是多少」
run启动项目「帮我把程序运行起来」
init初始化项目「帮我初始化 CLAUDE.md」
simplify简化代码「帮我把这段代码优化一下」
reviewPR 审查「帮我 review 这个 PR」

六、跟 Claude Code 说话的技巧
